📌 README - Viva Segura System
🚨 Sobre o Projeto

O Viva Segura System é um sistema desenvolvido em linguagem C com o objetivo de simular um aplicativo de segurança voltado para mulheres em situação de risco.

A proposta do projeto é criar uma solução que funcione de forma rápida, automática e eficiente, especialmente em momentos onde a vítima não tem tempo para interagir manualmente com o aplicativo.

🎯 Objetivo

Desenvolver um sistema capaz de:

Acionar emergência automaticamente
Enviar localização em tempo real
Simular integração com autoridades
Oferecer suporte preventivo (monitoramento e mapa de risco)
Aumentar a segurança da usuária

🧠 Conceito do Sistema

O sistema foi pensado com base em situações reais, onde:

A usuária pode estar em perigo
Não há tempo para abrir um aplicativo
A resposta precisa ser imediata

Por isso, o projeto inclui:

Ativação por botão físico (celular/relogio)
Ativação por comando de voz
Monitoramento automático
Sistema de medida protetiva inteligente

⚙️ Funcionalidades
👤 Cadastro
Dados completos da usuária (CPF, RG, telefone, endereço)
Contato de emergência
Informações opcionais:
Tipo sanguíneo
Alergias
Registro de medida protetiva

🚨 Emergência
Ativação automática (voz ou botão)
Envio de localização para:
Polícia
Ambulância
Contato de emergência
Simulação de direcionamento de câmeras da cidade
Vibração do dispositivo

📡 Monitoramento
Envio contínuo de localização
Acompanhamento em tempo real
Notificação ao contato de emergência
Integração com câmeras (simulada)

⚠️ Medida Protetiva
Sistema automático (sem ação da usuária)
Verificação de distância do agressor
Caso o limite seja ultrapassado:
Polícia é acionada automaticamente
Localização da vítima é enviada

🗺️ Mapa de Risco
Identificação de áreas perigosas
Solicita uso de localização
Sugere rota alternativa segura
Integração simulada com apps como Waze

📢 Denúncia
Registro de ocorrência
Possibilidade de anonimato
Descrição livre

🔒 Sistema de Permissões

Simula o comportamento de aplicativos reais:

Localização
Microfone (comando de voz)
Dispositivo (botões/relogio)

🧩 Estrutura do Código

O sistema foi desenvolvido utilizando:

struct → para organizar os dados da usuária
funções → para separar cada funcionalidade
switch → controle do menu
do-while → repetição do sistema

🛠️ Tecnologias Utilizadas
Linguagem C
Biblioteca stdio.h
Biblioteca string.h

💡 Diferenciais do Projeto
Ativação automática (sem interação direta)
Simulação de integração com cidade inteligente
Uso de comandos de voz
Sistema baseado em situações reais
Estrutura organizada e escalável

🚀 Possíveis Melhorias
Integração real com GPS
Uso de APIs de mapas (Google Maps, Waze)
Conexão com banco de dados
Interface gráfica (mobile)
Integração com dispositivos reais (smartwatch)

🏆 Conclusão

O Viva Segura System demonstra como a tecnologia pode ser utilizada para criar soluções que impactam diretamente a segurança e o bem-estar das pessoas.

Mesmo sendo uma simulação em linguagem C, o projeto apresenta conceitos modernos e aplicáveis ao mundo real.
